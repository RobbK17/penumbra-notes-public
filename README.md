# Penumbra Notes

**Private by default. Markdown at heart. Yours alone.**

Penumbra Notes is a native desktop app for macOS and Windows. Your notes live in a local database on your machine—optionally encrypted—with rich markdown editing, tags, tasks, wiki links, and an optional right-hand inspector (outline, statistics, version history, and more).

> **Beta software** — These builds are for beta testing. Expect rough edges, and back up your vault before upgrading. Releases here are installable builds for testers, not a promise of production stability.

## Download

## Highlights

- **Local-first** — SQLite vault on disk; optional passphrase encryption. No account required.
- **Markdown editor** — WYSIWYG writing with tasks, tables, images, Mermaid diagrams, callouts, and `[[wiki links]]`.
- **Organization** — Tags, sidebar filters (Todos, Types, Archive, Trash), search, and optional tabs.
- **Inspector** — Outline, statistics, version history, reminders, journal, and optional local AI (Ollama).
- **Optional sync** — Self-hosted backup and shared collections if you run your own sync server.

## First run

1. Install from the release asset for your platform.
2. **File → Create database…** (or open an existing vault).
3. Start writing. Notes autosave as you type.

## Privacy

Notes and vault data stay on your computer unless you explicitly enable sync or export. This repository distributes **binaries only**—no application source code and no telemetry bundled with downloads.
